Приложение для мониторинга чатов в Telegram. При обнаружении в сообщении ключевого слова/фразы, отправляется
ответ с промо-текстом.

Перед использованием нужно получить API_ID и API_HASH для доступа к чатам Telegram: https://my.telegram.org/auth?to=apps
Введите полученные API_ID и API_HASH, а также номер телефонав файл "api_id, api_hash, phone, password.txt"
(в папке с приложением) вместо звёздочек.

При первом запуске приложения придёт код подтверждения. Откройте файл code.txt и введите код вместо
звёздочек. Сохраните файл и закройте.

Чтобы в автоматическом режиме отвечать в чате, нужно сперва вступить в этот чат (с аккаунта,
который будет использоваться этим приложением).

Если мониториться будет "супергруппа", должна быть ссылка именно на эту группу, а не на её отдельную комнату.

Приложение автоматически проверяет ссылки на актуальность. Если какой-то чат был удалён,
он перестаёт отслеживаться.

Текст промо-сообщения (то, что будет отправляться в ответ на сообщения с ключевыми словами/фразами) выбирается случайным
образом из нескольких вариантов.

Управление происходит путём написания команд в сохранённые сообщения (папка Избранное).
Всего есть 8 команд:
+lnk - добавить ссылку/ки
-lnk - удалить ссылку/ки
+kw - добавить ключевое слово/фразу (можно сразу несколько)
-kw - удвлить ключевое слово/фразу (можно сразу несколько)
rt - установить диапазон времени ответа в секундах
+promo - добавить промо-сообщение
-promo - удалить промо-сообщение
info - показать информацию о текущих ключевых словах /фразах, отслеживаемых ресурсах, тексте промо-сообщения, времени ответа

Ссылки добавляются в список написанием команды +lnk (от англ. link). После команды следует
пробел, затем указывается одна или несколько ссылок. Если несколько, то они должны быть
разделены символом *. Например:
+lnk https://t.me/ByteBustersCoders*https://t.me/itexpress101*https://t.me/IlyDja_chat

Удаление ссылок происходит аналогично, но по команде -lnk

Ключевые слова/фразы добавляются в список написанием команды +kw (от англ. keyword). После команды следует
пробел, затем указывается одно или несколько ключевых слов/фраз. Если несколько, то они должны быть
разделены символом *. Например:
+kw нужен менеджер*продам гараж*возьму в аренду огурец

Удаление ключевых слов/фраз происходит аналогично, но по команде -kw

Время ответа устанавливается командой rt (от англ. reply time). После команды следует пробел, затем указывается
два числа - диапазон в секундах (включительно), из которого случайным образом каждый раз будет выбираться время, необходимое для
имитации набора текста. Например:
rt 10 30
По умолчанию установлен диапазон 5 20

Промо-сообщение добавляется командой +promo. Например:
+promo Здравствуйте, напишите в ЛС

Промо-сообщение удаляется командой -promo. Например:
-promo Здравствуйте, напишите в ЛС

Информацию о текущих ключевых словах /фразах, отслеживаемых ресурсах, тексте промо-сообщения/ий
можно получить командой info

Если обнаружите в списке отслеживаемых ресурсов рерурс под названием "me", не удивляйтесь - это
папка Избранное. Не удаляйте этот ресурс из отслеживаемых, иначе утратится возможность
управления скриптом через папку Избранное.

Администрация Телеграм может вводить различные ограничения (на частоту запросов, на количество отслеживаемых
ресурсов и т.п.). Точных ограничений не знает никто, кроме самой администрации Телеграм.

После запуска программа может быть в статусе "Не отвечает". Это нормально - не обращайте внимания. При закрытии 
программы Windows предложит подождать отклика - откажитесь (нажмите "Всё равно завершить")

Если нужно войти под другим аккаунтом или ошиблись при логине, завершите приложение,
удалите файл telethon_session.session

Если произошёл сбой, ищите в папке с приложением файл log.txt - отправьте его
разработчику для анализа.
